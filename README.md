# RozgarFind — Pakistan's Local Job Finder

A complete Android job portal application built for Pakistan that connects job seekers with local employers.

## About the App

RozgarFind is Pakistan's local job finder mobile application built using Java and Android Studio. It allows job seekers to find local jobs in 9 categories and apply directly from their mobile phone. Employers can post job listings and manage them through an admin account. The app solves the problem of lack of local job platforms for daily-wage and semi-skilled workers in Pakistan. It provides a simple, fast, and accessible platform for both job seekers and employers.

## Features

- User registration and secure login
- Admin login with special privileges
- Browse jobs in 9 categories
- Search jobs by title, company, or location
- View full job details
- Apply for jobs with personal details and cover letter
- Call employer directly from app
- Contact employer via WhatsApp
- Admin can post new job listings
- Admin can delete existing job listings
- View all submitted applications

## Technologies Used

- Java
- Android Studio
- XML Layouts
- SQLite Database
- Material Design

- Implicit Intents (Call + WhatsApp)

## How to Install the APK

1. Download the APK file from the link above.
2. Open the APK file on your Android phone.
3. If asked, allow installation from unknown sources in Settings.
4. Install and open RozgarFind app.

## How to Run the Project

1. Download or clone this repository.
2. Open Android Studio.
3. Click File → Open → Select RozgarFind folder.
4. Wait for Gradle sync to complete.
5. Connect Android phone or start emulator.
6. Click Run button.
